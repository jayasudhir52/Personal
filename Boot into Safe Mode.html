Boot into Safe Mode

Why Use Safe Mode?

  • Starts Windows with minimal drivers and services.
  • Helps diagnose software and driver issues.
  • Allows removal of problematic programs or malware.
  • Improves troubleshooting when normal boot fails.

Steps to Boot into Safe Mode (Windows)

  • Restart the computer.
  • Interrupt the boot process three times to access recovery mode.
  • Select Troubleshoot → Advanced options → Startup Settings.
  • Click Restart.
  • Press 4 or F4 for Safe Mode.
  • Press 5 or F5 for Safe Mode with Networking.

After Booting into Safe Mode

Once in Safe Mode, uninstall recently added software, update drivers, or run antivirus scans. Restart the system normally to check if the issue has been resolved.
