Test RAM and Motherboard

Test RAM

  • Power off the system and unplug it from the power source.
  • Remove all RAM sticks from the motherboard.
  • Insert a single RAM stick into the recommended slot.
  • Power on the system and check if it boots.
  • Test each RAM stick individually to identify faulty memory.
  • Clean RAM contacts gently and ensure they are seated firmly.

Test Motherboard

  • Check the motherboard for visible damage or burnt components.
  • Listen for beep codes or observe diagnostic LEDs during startup.
  • Reset the BIOS by clearing the CMOS.
  • Ensure the motherboard is not shorting against the case.
  • Test the system outside the case if necessary.
  • Use known-working RAM and PSU to confirm motherboard failure.

Final Recommendation

If the system fails to boot after testing RAM and verifying all connections, the motherboard may be defective and should be replaced with a compatible model.
